# Data-Driven Formula 1 — Big Data Management & Processing

**Course:** COMP-548DL Big Data Management and Processing  
**Student:** Theofanis Lymbouris (U241N1154)

## Project Overview
This project analyzes Formula 1 race strategy and driver performance using
Big Data management and processing techniques. Multi-season race data is
ingested into a NoSQL database and processed using Apache Spark (PySpark).

## Big Data Focus
- Primary dimension: Volume
- Secondary dimension: Velocity (simulated streaming)
- Data management: NoSQL (MongoDB)
- Processing engine: Apache Spark (PySpark)

## Repository Structure
- `src/ingest/` — data ingestion pipelines
- `src/processing/` — Spark batch & streaming jobs
- `notebooks/` — analysis & visualization
- `docs/` — figures & diagrams
